# CLI Reference

CodeTether command-line interface.

## Commands

### serve

Start the server:

```bash
codetether serve --port 8000
```

### worker

Start a worker:

```bash
codetether worker --server https://example.com
```

### version

Show version:

```bash
codetether --version
```
