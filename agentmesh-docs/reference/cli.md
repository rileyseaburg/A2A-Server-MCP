# CLI Reference

AgentMesh command-line interface.

## Commands

### serve

Start the server:

```bash
agentmesh serve --port 8000
```

### worker

Start a worker:

```bash
agentmesh worker --server https://example.com
```

### version

Show version:

```bash
agentmesh --version
```
