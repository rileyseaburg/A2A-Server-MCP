# Agent Messaging

AgentMesh enables agents to communicate and collaborate using the A2A Protocol.

## Overview

Agents can:

- **Discover** other agents via Agent Cards
- **Send tasks** to other agents
- **Receive results** via streaming or polling
- **Collaborate** on complex workflows

See [A2A Protocol](../concepts/a2a-protocol.md) for details.
